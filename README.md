## 🚀 How to Run

### Prerequisites
```bash
pip install pandas numpy
```

### Task 1 — Rule-Based Chatbot
```bash
python task1_chatbot.py
```
**Features:**
- Regex-based pattern matching
- Responds to greetings, jokes, AI questions, Python topics & more
- Random response selection for variety

---

### Task 2 — Tic-Tac-Toe AI
```bash
python task2_tictactoe.py
```
**Features:**
- Minimax algorithm — AI is unbeatable
- Human plays as `X`, AI plays as `O`
- Play again option after each game

---

### Task 4 — Movie Recommendation System
```bash
python task4_recommendation.py
```
**Features:**
- **Content-Based Filtering** — recommends movies similar in genre
- **Collaborative Filtering** — recommends based on similar users' preferences
- Dataset of 15 popular movies with genres and ratings
